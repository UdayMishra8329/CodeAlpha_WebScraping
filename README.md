# BBC News Web Scraping and Data Analysis

## Project Overview

This project is a complete web scraping and data analysis solution built using Python. It collects news articles from BBC News, stores the data in multiple formats, performs data cleaning, and generates visual insights through charts and statistical analysis.

The project demonstrates web scraping, data preprocessing, data visualization, and report generation skills.

---

## Features

* Scrapes latest news articles from BBC News
* Collects article title, URL, category, date, and summary
* Removes duplicate records
* Stores data in CSV, JSON, and Excel formats
* Generates statistical summaries
* Creates multiple visualizations using Matplotlib
* Produces a final analysis report
